# Blog Images

Place blog post images in this directory.

## Usage

When creating a new blog post, reference images using:

```markdown
image: "/images/blog/your-image-name.jpg"
```

## Supported formats

- `.jpg` / `.jpeg`
- `.png`
- `.webp`
- `.svg`

## Naming convention

Use descriptive, lowercase names with hyphens:
- `my-first-post.jpg`
- `data-visualization-example.png`
- `microbiome-analysis-diagram.svg`
